# 🐾 Waffle - Tu Mascota Virtual en la Terminal

¡Bienvenido a **Waffle**, un juego estilo Tamagotchi completamente desarrollado en Python que corre directamente en tu terminal! Cuida de Waffle, mantenlo alimentado, juega con él y vigila que no se quede sin energía.

---

## 🚀 Características

* **Mascota Interactiva:** Responde en tiempo real a tus acciones.
* **Arte ASCII:** ¡Un diseño retro en texto para ver el estado de salud de Waffle!
* **Mecánicas de Estado:** Gestión de `energía` y `felicidad` basada en tus decisiones.
* **Bucle de Juego:** El juego continúa de forma dinámica hasta que la energía de Waffle se agota por completo.

---

## 🛠️ Acciones Disponibles

En cada turno podrás elegir una de las siguientes opciones para interactuar con Waffle:

1.  **Alimentar (🍎):** Aumenta la energía (+20) y un poco la felicidad (+1).
2.  **Jugar (⚽):** Aumenta mucho la felicidad (+20), pero consume energía (-15).
3.  **Ver estado de salud (📊):** Muestra el arte ASCII de Waffle y sus estadísticas actuales.
4.  **No hacer nada (💤):** Waffle se aburre. Disminuye la felicidad (-10) y la energía (-5).
